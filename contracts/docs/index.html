Immutable Index Token

Monad Meme Top 10 Index (TOP10)

TOP10 is a permissionless, immutable index token representing a fixed basket of 10 Monad meme tokens. Minting/redeeming uses the underlying constituents—no owner, no admin keys.

How it works
  • Mint: deposit the 10 tokens in the fixed basket proportions to receive TOP10.
  • Redeem: burn TOP10 to receive the underlying tokens back.
  • No auto-rebalance: basket is fixed (immutable); composition does not change.
